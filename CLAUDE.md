# Plant Blog - Claude Instructions

## プロジェクト概要

Astro 製の静的サイト。コンテンツは Scrapbox (Cosense) API からビルド時に取得。
Cloudflare Pages でホスティング、画像は Cloudflare R2。
ドメイン: tomo-at-plants.com

## コード変更時のレビュー

**.astro ファイルを変更・追加したときは、以下を必ずチェックして問題があれば報告または修正する。**

### 共通化チェック

- 複数の `.astro` ファイルに同じ CSS が重複していないか
- コンポーネントに切り出せる共通構造（ナビ、パンくず、カードなど）がないか
- JS ロジックが複数箇所に散らばっていないか

### アクセシビリティ（JIS X 8341-3:2016 AA 基準）

- `<img>` に `alt` 属性があるか
- 色のコントラスト比が通常テキスト 4.5:1 以上、大きいテキスト 3:1 以上か
- キーボード操作（Tab キー移動・Enter/Space 実行）が可能か
- 見出し階層（h1 → h2 → h3）が論理的に正しいか
- リンクテキストが文脈なしで意味をなすか（「こちら」「詳細」のみは NG）
- フォームにラベルがあるか（フォームを追加した場合）
- `lang` 属性が HTML に設定されているか

問題が見つかった場合は修正案を提示し、ユーザーの確認を取ってから変更する。
問題がなければ「レビュー: 問題なし」と一言添える。
